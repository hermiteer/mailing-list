MailingList depends on two files from Apple's Reachability sample: Reachability.h and Reachability.m.

Simply get the Reachability sample from the Apple Developer website link below, copy those two files into the Reachability folder inside of this project's root directory and you're ready to go.

https://developer.apple.com/Library/ios/samplecode/Reachability/Introduction/Intro.html
